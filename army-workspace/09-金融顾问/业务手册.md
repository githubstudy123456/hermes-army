# 金融顾问部 Financial Advisory

## 业务定位
为企业、投资人提供 AI 辅助的财务分析与投资顾问服务。

## 可开展业务

### 1. 股权研究（现有 cron 业务）
- 每日一只A股深度研究
- 行业赛道分析
- 竞争对手对比

### 2. 财务建模
- 可比公司分析（Comps）
- DCF 现金流折现模型
- LBO 杠杆收购模型
- 财务尽调

### 3. 宏观经济监测
- A股大盘 + 汇率实时监控
- 央行政策解读
- 美联储动态追踪

### 4. 融资支持
- 商业计划书（BP）财务章节
- 估值分析
- 投资人尽调材料支持

### 5. 个人财务顾问
- 资产配置建议
- 基金/股票筛选逻辑输出

## 执行步骤
1. 需求确认 → 研究报告 / 建模 / 监测
2. 数据收集 → 财报 / 招股书 / 宏观数据
3. AI 分析 → 输出初稿
4. 专业审核 → 修正结论
5. 交付 + 定期更新（监测类服务）
