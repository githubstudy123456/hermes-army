# 内容创作部 Content Studio

## 业务定位
为企业、个人品牌提供 AI 驱动的内容生产服务。

## 可开展业务

### 1. 品牌内容矩阵
- 微信公众号 / 小红书 / 知乎代运营
- 每周内容日历 + AI 生成初稿 + 人工审核
- 按篇收费或月度打包

### 2. 视频脚本 & 短视频批量生产
- 行业知识类短视频脚本（60s 内）
- 产品介绍 / 种草 / 教程类脚本
- 提供分镜建议 + 字幕文本

### 3. 新闻通稿 & 品牌文案
- 企业融资发布、战略合作公告
- 产品上市通稿
- 创始人专访背景材料

### 4. SEO 内容工厂
- 目标关键词研究
- 长文 SEO 优化内容批量生产
- 行业白皮书 / 报告

### 5. K-pop / 泛娱乐资讯推送（现有 cron 业务）
- 娱乐媒体监测
- 飞书群定时推送

## 执行步骤
1. 客户需求对接 → 填写需求单
2. 关键词 / 选题策划
3. AI 生成初稿 → 内容审核
4. 客户确认 → 交付
5. 效果追踪（阅读量 / 转化）
