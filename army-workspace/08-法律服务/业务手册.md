# 法律服务部 Legal Services

## 业务定位
为企业及高净值个人提供 AI 辅助的法律合规服务。

## 可开展业务

### 1. 合同审查
- NDA / 保密协议
- 供应商采购合同
- SaaS 服务协议（MSA）
- 劳动合同合规审查

### 2. 监管合规
- 行业合规评估（数据、金融、医疗）
- 监管政策解读报告
- 算法备案 / 生成式 AI 合规

### 3. 知识产权
- 商标监测与预警
- 软件著作权登记材料
- 开源许可合规审查

### 4. 企业法律顾问
- 常年法律咨询（按小时或包年）
- 董事会会议记录合规
- 商业尽调法律支持

## 执行步骤
1. 需求分类 → 合同类型 / 合规领域
2. 材料收集 → 合同文本 / 营业执照 / 资质文件
3. AI 初审 → 风险点标注 + 修改建议
4. 律师复核 → 出具正式意见
5. 客户确认 → 交付签署
