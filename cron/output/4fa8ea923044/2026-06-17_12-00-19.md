## 一、饮食管理（life-diet）

### 核心功能
- 营养建议：根据目标（减脂/增肌/维持）提供热量和 macronutrient 建议
- 餐食规划：每日/每周餐食计划
- 饮食记录分析：分析饮食日志，找出改进点

### 常用数据

**基础代谢估算（BMR， Mifflin-St Jeor）：**
- 男性：BMR = 10×体重(kg) + 6.25×身高(cm) - 5×年龄 + 5
- 女性：BMR = 10×体重(kg) + 6.25×身高(cm) - 5×年龄 - 161

**每日总消耗（TDEE）= BMR × 活动系数：**
| 活动水平 | 系数 | 描述 |
|----------|------|------|
| 久坐 | 1.2 | 办公室工作，无额外运动 |
| 轻度 | 1.375 | 每周1-3天轻度运动 |
| 中度 | 1.55 | 每周3-5天中等运动 |
| 高强度 | 1.725 | 每周6-7天高强度运动 |
| 运动员 | 1.9 | 每天2次高强度训练 |

### 目标与热量对照
| 目标 | 每日赤字/盈余 | 每周变化 |
|------|--------------|----------|
| 减脂 | -500 kcal | -0.5kg/周 |
| 维持 | 0 | 0 |
| 增肌 | +300~500 kcal | +0.5kg/周 |

### SMART 目标模板
```
S - Specific（具体的）：我要学什么？

A - Achievable（可实现的）：现有资源够吗？
R - Relevant（相关的）：为什么现在重要？
T - Time-bound（有时限的）：截止日期是？
```

## 通用原则
