## 3. 3D场景设计规范

### 3.1 模型制作规范
```json
{
  "format": "glTF 2.0 / glB",
  "scale": "1 unit = 1 meter",
  "max_vertices": 50000,
  "max_texture_size": 2048,
  "compression": "Draco"
}
```

### 3.2 材质规范
- PBR材质优先
- 颜色区分不同部件
- 金属/塑料/玻璃材质区分

### 3.3 交互设计
- 旋转：鼠标左键拖拽
- 缩放：鼠标滚轮
- 平移：鼠标右键拖拽
- 重置：双击

## 4. 技术实现方案

### 4.1 核心代码结构
```typescript
// 3D场景管理
class Physics3DScene {
  scene: THREE.Scene;
  camera: THREE.PerspectiveCamera;
  renderer: THREE.WebGLRenderer;
  
  // 加载模型
  async loadModel(url: string): Promise<THREE.Group>;
  
  // 设置视角
  setView(view: 'front' | 'side' | 'top' | 'custom');
  
  // 动画控制
  playAnimation(name: string);
  stopAnimation();
}

// 实验场景
class ExperimentScene extends Physics3DScene {
  // 实验特定逻辑
  recordData(data: any);
  generateReport();
}
```

### 4.2 性能优化策略
- 模型预加载
- LOD (Level of Detail)
- 视锥剔除
- 帧率控制

### 4.3 移动端适配
- 点触控交互
- 简化模型版本
- 降低渲染质量

### 5.2 API接口
```
GET /api/v1/3d-models/:id        # 获取模型信息
GET /api/v1/3d-models/:id/scene   # 获取场景配置
POST /api/v1/3d-models/upload    # 上传模型
```
