## 各部门交付物摘要

### 1. lobster-product → 物理科目PRD
**核心内容：**
- 页面结构：首页 → 学科选择 → 物理科目首页 → 知识模块/视频/实验/习题
- 功能模块：知识库、视频课程、实验中心、习题练习
- 物理特有：公式展示LaTeX、图表可视化、3D模型、数据分析
- 验收标准：产品经理强制审查

### 2. lobster-architect → 系统架构设计
**核心内容：**
- 技术栈：React/Vue + Node.js + PostgreSQL + Redis + Three.js
- 架构模式：前后端分离 + 微服务架构
- 多学科扩展：模块化插件设计，统一接口规范
- 核心服务：用户服务、内容服务、学习服务、3D服务

### 3. lobster-dev-frontend → 物理页面UI优化
**核心内容：**
- 页面布局优化：侧边栏章节树 + 内容区
- 公式展示：LaTeX渲染优化
- 通用组件：Button、Card、Navigation、Form、Table
- 性能优化：路由懒加载、图片懒加载、虚拟列表

### 4. lobster-dev-backend → 内容管理API设计
**核心内容：**
- API规范：RESTful + 统一响应格式 + JWT认证
- 核心接口：
  - 学科管理：GET /subjects, GET /subjects/:id
  - 知识点：CRUD全功能
  - 视频内容：上传、播放
  - 学习进度：记录和报告
- 错误码定义完整

### 5. lobster-dev-3d → 物理3D模型方案
**核心内容：**
- 技术选型：Three.js + Blender + GLTF/GLB
- 模型分类：力学(5)、热学(3)、光学(4)、电磁学(5)、近代物理(3)
- 场景规范：模型制作规范、材质规范、交互设计
- 开发计划：3阶段递进

### 6. lobster-test → 测试计划
**核心内容：**
- 测试用例：学科模块、视频课程、3D模型、习题模块
- 跨学科扩展测试策略
- 性能指标：页面<3s、视频<2s、3D<3s、API<200ms
- 安全测试要点：SQL注入、XSS、CSRF
- 验收标准：产品经理签字确认
