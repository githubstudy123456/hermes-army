## 4. 性能测试

### 4.1 性能指标
| 指标名称 | 目标值 | 临界值 |
|----------|--------|--------|
| 页面加载时间 | < 3s | < 5s |
| 首屏渲染 | < 1.5s | < 2s |
| 视频播放启动 | < 2s | < 3s |
| 3D模型加载 | < 3s | < 5s |
