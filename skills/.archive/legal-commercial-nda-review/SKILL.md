# NDA Review · 保密协议快速分类

## 核心定位

接收到 NDA 文本（PDF、DOCX 或粘贴内容），在 2 分钟内完成 **GREEN / YELLOW / RED** 三级分类，并输出结构化风险摘要。  
输出为**草稿**，需律师审核确认，不构成法律结论。

---

## 分类标准

### GREEN — 低风险（标准条款，可直接签署）

| 判断条件（满足任意一条） |
|------------------------|
| 双向 NDA，且保密期限 ≤ 2 年 |
| 保密信息定义合理，无过度扩展 |
| 无单方面修改权 |
| 无不合理竞业/禁止招揽条款 |
| 违约救济限于实际损失，不含惩罚性赔偿 |
| 适用法律/管辖法院为中国大陆 |

**GREEN 输出模板：**
```
🟢 GREEN — 低风险

合同类型：双向 NDA
对方当事人：[甲方/乙方]
保密期限：[X] 年

【检查结果】
✅ 保密信息定义合理
✅ 披露限制标准（"善意管理"即可）
✅ 无单方面修改权
✅ 违约救济对等（实际损失赔偿）
✅ 适用法律明确（中国大陆）

【风险摘要】无重大风险点

【建议】可直接签署，或按需调整保密期限
```

---

### YELLOW — 中等风险（需关注，部分条款建议谈判）

| 判断条件（满足任意一条） |
|------------------------|
| 单向 NDA（一方仅承担义务，无对等权利） |
| 保密期限 > 3 年 |
| 保密信息定义过宽（包含"一切信息"等表述） |
| 存在自动续约条款（无明确终止通知机制） |
| 违约救济含惩罚性赔偿（超出实际损失） |
| 适用法律/管辖法院为境外 |
| 包含 IP 归属约定但不明确 |
| 限制性条款（禁止招揽、竞业限制）超出合理范围 |

**YELLOW 输出模板：**
```
🟡 YELLOW — 中等风险

合同类型：[单/双向] NDA
对方当事人：[甲方/乙方]
保密期限：[X] 年

【高风险条款】
1. [条款位置 + 内容摘要]
   风险：xxxxx
   建议修改：xxxxx

2. ...

【中等风险条款】
1. ...

【建议】
- 必须谈判修改：xxxxx
- 建议谈判优化：xxxxx
- 可接受条款：xxxxx

【律师介入】建议法务同事复核后再签署
```

---

### RED — 高风险（重大风险，建议拒绝或重大修改）

| 判断条件（满足任意一条） |
|------------------------|
| 无保密期限限制（永久有效） |
| 保密信息定义极度宽泛（含"合作本身"等） |
| 包含禁止招揽条款且限制期 > 2 年 |
| 单方面赋予对方修改协议权利 |
| 违约救济含三倍赔偿或律师费转移 |
| 适用法律为对中国当事人不利的境外法域 |
| 包含强制仲裁条款且仲裁地为境外 |
| 放弃一切法律救济权利的条款 |
| 强制要求转让 IP 所有权 |

**RED 输出模板：**
```
🔴 RED — 高风险

合同类型：[单/双向] NDA
对方当事人：[甲方/乙方]
保密期限：[X] 年

【高风险条款】（需立即处理）
1. [条款位置 + 内容摘要]
   风险等级：🔴高
   潜在后果：xxxxx
   建议：拒绝接受 / 要求删除

2. ...

【致命条款】
⚠️ [最危险的条款摘要] — 此条款若签署将导致 [具体风险]

【建议】
- 拒绝签署当前版本
- 重新谈判以下条款：xxxxx
- 或建议更换为公司标准 NDA 模板

【律师介入】必须由律师审核后方可决定是否谈判
```

## 工作流程

```
Step 1：读取合同
  ↓ 调用 file tool 读取 PDF/DOCX
  ↓ 或直接处理粘贴文本
Step 2：识别 NDA 类型
  ↓ 单向 vs 双向、保密期限、定义范围
Step 3：逐条扫描 RED 指标
  ↓ 若命中任何 RED 指标 → 直接输出 RED
Step 4：逐条扫描 YELLOW 指标
  ↓ 若命中 YELLOW 指标但无 RED → 输出 YELLOW
Step 5：无 RED/YELLOW → 输出 GREEN
Step 6：输出结构化报告
  ↓ 标注 [verify] 的条款需人工核实
```

## 关键检查清单

| 序号 | 检查项 | GREEN | YELLOW | RED |
|------|--------|-------|--------|-----|
| 1 | NDA 类型 | 双向 | 单向 | — |
| 2 | 保密期限 | ≤ 2年 | 3-5年 | 永久或 > 5年 |
| 3 | 保密信息定义 | 合理限定 | 过宽 | 极度宽泛 |
| 4 | 单方面修改权 | 无 | 存在 | — |
| 5 | 违约救济 | 实际损失 | 惩罚性 | 三倍/律师费转移 |
| 6 | 适用法律 | 中国大陆 | 境外但合理 | 对我不利的境外 |
| 7 | 竞业/禁止招揽 | 无 | 有限制但合理 | 超出合理范围 |
| 8 | IP 归属 | 不涉及或明确 | 有约定但不明确 | 要求强制转让 |

## 注意事项

- **引用来源标注 `[verify]`**：发现风险条款时，标注 `[verify]` 并说明"需人工核实"，防止错误 waived 律师-委托人特权
- **输出标注**：所有输出必须注明"草稿，需律师审核确认"
- **不承诺结果**：不保证分类绝对准确，仅提供参考意见
- **consequential actions**：涉及发送、归档、执行等操作时需明确用户确认
