# Diligence Issue Extraction 尽职调查问题提取技能

## name
legal-corporate-diligence-issue

## description
对尽职调查（DD）中发现的法律问题进行系统化分类、优先级评估及风险定级，输出结构化问题清单，适用于法律尽调、财务尽调及业务尽调场景，支持 red-flag / amber-flag / green-flag 分级。

## trigger
- 用户上传尽调报告或访谈记录文本
- 用户询问"尽调问题"、"风险点"、"合规瑕疵"等
- 项目立项或投资决策阶段需要梳理风险清单

## steps
1. **接收尽调材料**：读取尽调报告、访谈纪要或问题清单文件
2. **识别问题类别**：股权结构、知识产权、劳动人事、诉讼仲裁、环保合规、税务等
3. **提取问题要点**：问题描述、涉及主体、涉及金额（如有）、发现来源
4. **进行风险定级**：Red（重大障碍）/ Amber（需关注）/ Green（轻微或无实质影响）
5. **建议处理方案**：披露/整改/赔偿承诺/放弃交易等
6. **输出结构化问题清单**：按优先级排序，含问题描述、风险等级、建议措施

## pitfalls
- 将法律与商业问题混淆，未区分纯法律风险
- 对 red-flag 问题未深入核查即建议"带病推进"
- 忽视地方性法规与中央法规的差异（如环保地方标准）
- 问题定级缺乏量化依据（如涉及金额、影响人数）

## verification
- 每项问题均有风险等级标注及定级理由说明
- 涉及金额或人数的问题已量化呈现
- 同类问题已归类汇总，避免重复列举
- 可通过 file tool 读取历史尽调报告验证分级标准一致性
