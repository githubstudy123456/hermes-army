# 个人成长技能

## 核心功能

1. **目标制定** — 协助制定可执行的成长目标
2. **读书计划** — 选书建议、读书笔记整理、阅读进度追踪
3. **学习记录** — 每日学习内容记录、周/月总结
4. **技能追踪** — 新技能学习进度、可视化进展

---

## 目标制定框架（SMART）

每个目标必须符合：
- **S**pecific（具体）
- **M**easurable（可衡量）
- **A**chievable（可达成）
- **R**elevant（相关）
- **T**ime-bound（有期限）

例行目标格式：
```
【成长目标】目标名称

具体描述：
达成标准：
截止日期：
为什么重要：

分解到每周：
- Week 1：[具体行动]
- Week 2：[具体行动]
- ...

需要的资源/支持：
可能的障碍：

完成后如何庆祝：
```

## 读书计划模板

```
【读书计划】《书名》

阅读速度：[每天X页/每周X章]
开始日期：
预计完成日期：
当前页数：X / 总页数 X

每周目标：
- Week 1：第X-X章 — 核心问题：[列出章节主题]
- ...

读后感/笔记模板：
## 读书笔记 | 《书名》| 第X章

本章核心观点：
我的理解：
可以应用的场景：
金句摘录：
```

## 成长周报/月报模板

```
【成长周报】YYYY年第XX周

📚 阅读/学习：
- [书名/课程] — 进度 X%

💪 技能进展：
- [技能A] — [进度]
- [技能B] — [进度]

🎯 目标达成：
- [目标1] — ✅达成 / 🔄进行中 / ⚠️落后

💡 本周最大收获：
[1-2句话总结]

📅 下周重点：
1. [具体行动]
2. [具体行动]
```

---

## 输出风格

鼓励为主，认可每一个小进步。不施压，用进度可视化和正向反馈驱动。
