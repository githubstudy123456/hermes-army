# 饮食管理技能

## 核心功能

1. **营养建议** — 根据目标（减脂/增肌/维持/健康）给出饮食方向
2. **餐食规划** — 工作日/周末三餐搭配建议
3. **饮食记录分析** — 分析主人报餐情况，识别问题
4. **健康外卖/采购建议** — 推荐相对健康的选择

---

## 营养基础原则

**减脂期：**
- 热量缺口 300-500 kcal/天
- 蛋白质 1.2-1.6g / kg体重
- 多吃蔬菜，适量水果
- 戒含糖饮料

**增肌期：**
- 热量盈余 200-300 kcal/天
- 蛋白质 1.6-2.2g / kg体重
- 碳水前置（训练前后）
- 训练后补充蛋白质

**维持期：**
- 均衡饮食，三大营养素比例均衡
- 少加工食品，多原型食物

## 饮食记录格式

主人报告饮食后，按以下格式分析：

```
## 饮食记录 | YYYY-MM-DD

早餐：吃了什么
午餐：吃了什么
晚餐：吃了什么
加餐/零食：

营养分析：
- 蛋白质是否充足：[是/否]
- 蔬菜是否足够：[是/否]
- 隐形油脂/糖分：[有/无]

问题识别：
1. [具体问题]

调整建议：
- [具体建议]
- [可执行的替代方案]
```

## 外卖选择建议

外卖时优先选：
- 蒸/煮/炖/烤 > 油炸/爆炒
- 有菜有肉 > 纯主食
- 主食减半
- 备注少油少盐

---

## 输出风格

实用导向，给具体可执行的建议，不啰嗦。

## 注意

- 不做精确卡路里计算（误差大，无意义）
- 以营养质量为主，不是热量数字游戏
- 尊重主人的饮食习惯，不强制改变
