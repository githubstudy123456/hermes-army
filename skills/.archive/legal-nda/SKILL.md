# 定制化NDA生成器

## 触发条件

- 用户说"生成NDA"、"写个保密协议"、"帮我起草保密协议"
- 用户描述合作场景并要求"签个协议"

## 输入信息收集

如果信息不全，按需询问：

| 信息项 | 说明 | 默认值 |
|--------|------|--------|
| 甲方（披露方） | 全称 | 需提供 |
| 乙方（接收方） | 全称 | 需提供 |
| NDA类型 | 单向/双向/员工版 | 双向 |
| 合作目的 | 描述合作内容 | 需提供 |
| 保密期限 | | 2年 |
| 保密义务存续期 | 协议终止后 | 3年 |
| 适用法律 | 管辖法院 | 中国大陆 |

## 输出内容

生成一份完整NDA，包含：
1. 协议双方及背景
2. 保密信息定义（根据目的定制范围）
3. 保密义务（使用限制、接触人员范围、保护标准）
4. 例外情形（公知信息/独立开发/法律强制披露等）
5. 保密期限与终止
6. 信息返还/销毁
7. 违约救济（赔偿+律师费）
8. 无许可/无担保声明
9. 完整协议条款
10. 双方签署栏

**每一条款附带通俗解释**，方便非法律专业用户理解。

## 注意

- 中文合同，不使用英文法律术语
- 违约赔偿部分说明"实际损失+合理律师费"
- 如涉及知识产权，明确约定归属
