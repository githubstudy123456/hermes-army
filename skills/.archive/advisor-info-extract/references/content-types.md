# 信息提炼参考手册

收录主人发来的各种信息类型，以及对应的提炼模式。

---

## 已验证的内容类型

### 长文章/新闻
- 提炼：一句话总结(TL;DR) + 关键事实 + 后续影响 + 对主人的价值

### 合同文本
- 提炼：合同类型 + 核心条款 + 高风险条款
- 转交：如需深度风险分析 → legal-risks

### 群消息/会议记录
- 提炼：共识 + 待办(Owner+Deadline) + 分歧点 + 悬而未决

### 数据报表
- 提炼：核心数字(vs上期/同期) + 异常波动 + 趋势 + 风险指标

### 代码/技术文档
- 提炼：功能 + 技术路径 + 限制/风险
- 转交：如需代码实现 → dev-director

## 主人偏好

- **直接看结果**，不问文件直接展示内容
- 结论先行，不要铺垫
- 提炼报告末尾问："还有其他信息我应该考虑吗？"
