# thesis-tracker · 投资主题追踪与验证

## 什么时候用

用户要求"追踪XX的投资逻辑"、"监控持仓主题兑现"、"帮我过一遍持仓的 thesis"时，触发本 skill。

## 核心职责

1. 梳理持仓股票的核心投资主题（Thesis）
2. 建立主题兑现的关键指标（KPI）监控列表
3. 追踪指标变化，判断主题是否强化或弱化
4. 触发条件：关键指标大幅偏离预设阈值时推送提醒
5. 输出主题追踪报告，支持 thesis 强化、维持或推翻的判断

## 触发条件

```
用户说"追踪茅台的投资逻辑"、"帮我过一遍持仓 thesis"、"主题兑现得怎么样了"
```

## 工作流程

### Step 1 — 梳理投资 Thesis

为每只持仓股票建立 Thesis 档案：

```markdown
## {股票} 投资主题档案

### 核心 Thesis
（1-3句话概括投资逻辑）

### 关键假设
| 假设 | 预期 | 监控指标 | 阈值 |
|------|------|---------|------|
| 假设1 | XX | KPI1 | ±X% |
| 假设2 | XX | KPI2 | ±X% |

### 催化剂清单
| 催化剂 | 预期时间 | 状态 |
|--------|---------|------|
| 催化剂1 | X月 | 已兑现/进行中/待验证 |

### 风险因素
- 风险1
- 风险2
```

### Step 2 — 定义监控指标

根据 Thesis 分解关键监控指标（每只股票 3-7 个）：

| 股票类型 | 典型监控指标 |
|---------|------------|
| 消费（茅台） | 批条价格、动销数据、库存周期、居民消费支出 |
| 科技（腾讯） | 游戏流水、广告收入、To B收入增速、利润率 |
| 银行（招行） | NIM、不良率、拨备覆盖率、存款增速 |
| 新能源（宁德） | 装机量、产能利用率、原材料成本、单价 |

### Step 3 — 抓取最新数据

使用 `browser` 工具从东方财富/巨潮获取数据：

```
东方财富行情页：https://em.eastmoney.com/gpzy/dashboard/{stockCode}
东方财富研报页：https://em.eastmoney.com/research/list?stockCode={stockCode}
```

### Step 4 — 指标对比与判断

```javascript
// 提取监控指标
const metrics = {
  currentValue: 32.5,  // 当前值
  priorValue: 30.1,    // 上期值
  threshold: 0.05,     // 5% 偏离阈值
  change: (32.5 - 30.1) / 30.1  // 变化率
};

const status = Math.abs(change) > threshold ? 'ALERT' : 'NORMAL';
JSON.stringify({ ...metrics, status });
```

### Step 5 — Thesis 状态判断

| 状态 | 定义 | 行动 |
|------|------|------|
| **强化** | ≥3个关键指标朝有利方向 | 维持/加仓考虑 |
| **维持** | 关键指标基本符合预期 | 持续监控 |
| **弱化** | ≥3个关键指标朝不利方向 | 减仓考虑 |
| **待验证** | 催化剂尚未兑现 | 等待下一催化剂节点 |

### Step 6 — 输出追踪报告

```markdown
## {股票} 主题追踪报告 · {日期}

### Thesis 状态：{强化/维持/弱化/待验证}

### 关键指标追踪
| 指标 | 上期 | 本期 | 变化 | 阈值 | 状态 |
|------|------|------|------|------|------|
| KPI1 | XX | XX | +X% | ±5% | ✅正常/⚠️偏离 |
| KPI2 | XX | XX | -X% | ±5% | ✅正常/⚠️偏离 |

### 催化剂进度
- [x] 催化剂1（已兑现）
- [ ] 催化剂2（进行中，X%完成）
- [ ] 催化剂3（待验证，X月）

### 综合判断
（基于指标和催化剂的综合分析）

### 风险提示
（仅供参考，不构成投资建议）
```

## 已知陷阱

1. **指标与 Thesis 错配** — 确保每个 KPI 直接支撑 Thesis，不是泛泛的财务指标
2. **阈值设置过宽/过严** — 5%-10% 是常见参考范围，根据股票特性调整
3. **单一指标主导** — 避免被一个指标带偏，综合判断更可靠
4. **时间维度忽略** — 某些指标是季度数据，不要用日频数据做判断
5. **所有输出标注"仅供参考，不构成投资建议"**

## 验证方法

- [ ] 每只持仓股票都有 Thesis 档案
- [ ] 每只股票的 KPI 数量 ≥3 且 ≤7
- [ ] 最新指标数据已获取并记录日期
- [ ] Thesis 状态判断有数据支撑
- [ ] 追踪报告已输出并记录
