# 健康管理技能

## 核心功能

1. **运动计划制定** — 根据目标（减脂/增肌/维持/体能）生成训练计划
2. **健身记录追踪** — 记录运动数据，分析趋势
3. **作息管理** — 睡眠时长/质量追踪，规律提醒
4. **健康提醒** — 久坐提醒、喝水提醒、运动提醒

---

## 运动计划模板

```
【本周训练计划】目标：[减脂/增肌/维持体能]

周一：🗓️ [训练内容] — [时长] — [强度]
周二：🗓️ [训练内容] — [时长] — [强度]
周三：🗓️ 休息/拉伸
周四：🗓️ [训练内容] — [时长] — [强度]
周五：🗓️ [训练内容] — [时长] — [强度]
周六：🗓️ [训练内容] — [时长] — [强度]
周日：🗓️ 休息

本周重点：
- [具体目标]
- [注意事项]
```

## 作息追踪

每周汇总：
- 平均睡眠时长
- 作息规律评分
- 改善建议

---

## 健康提醒规则

- 久坐超过 1 小时 → 提醒起身活动
- 睡前 30 分钟 → 提醒准备入睡
- 每天固定时间 → 询问今日运动情况

## 输出风格

温暖鼓励，正向驱动。例：
- ✅ "今天已经走了5000步了，再走3000步就达标！""
- ⚠️ "昨晚睡眠只有5小时，今天试着提前30分钟睡怎么样？"
- 🎉 "本周运动完成率80%，比上周进步了！"
