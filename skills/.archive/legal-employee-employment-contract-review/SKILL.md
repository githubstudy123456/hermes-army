# Employment Contract Review · 劳动合同审查

## 核心定位

接收到劳动合同文本（PDF、DOCX 或粘贴内容），完成**深度条款审查**，识别必备条款缺失、违法条款、风险分级，输出修改建议。  
覆盖劳动合同法规定的法定条款、各类合同类型（固定期限/无固定期限/完成一定工作任务）。  
输出为**草稿**，需律师审核确认，不构成法律结论。

---

## 合同类型

| 类型 | 适用场景 | 法定要求 |
|------|---------|---------|
| 固定期限劳动合同 | 约定确定终止时间的劳动合同 | 续签时注意连续订立问题 |
| 无固定期限劳动合同 | 无确定终止时间的劳动合同 | 连续工作满10年需订立 |
| 完成一定工作任务劳动合同 | 以完成特定任务为期限 | 任务完成后合同终止 |

---

## 法定必备条款（《劳动合同法》第17条）

| 条款 | 是否必备 | 常见问题 |
|------|---------|---------|
| 用人单位名称、住所、法定代表人 | ✅ 必备 | 信息不完整 |
| 劳动者姓名、住址、身份证件 | ✅ 必备 | 信息不完整 |
| 合同期限 | ✅ 必备 | 期限约定不明 |
| 工作内容（岗位/职位） | ✅ 必备 | 岗位描述过于模糊 |
| 工作地点 | ✅ 必备 | 地点过于宽泛（如"全国"） |
| 工作时间和休息休假 | ✅ 必备 | 与法定不符 |
| 劳动报酬 | ✅ 必备 | 报酬约定不明或低于法定 |
| 社会保险 | ✅ 必备 | 未写明或约定不买 |
| 劳动保护、劳动条件和职业危害防护 | ✅ 必备 | 条款缺失 |
| 法律、法规规定应当纳入劳动合同的其他事项 | ✅ 必备 | 工伤/保密/竞业等 |

## 重点审查条款

### 1. 劳动期限

| 检查项 | 风险点 |
|-------|--------|
| 试用期期限 | 劳动合同期限<1年：≤1个月；1≤期限<3年：≤2个月；≥3年/无固定：≤6个月 |
| 试用期工资 | 不得低于正式工资80%，不得低于当地最低工资 |
| 试用期解除 | 需证明不符合录用条件，不能随意解除 |
| 连续订立次数 | 连续两次订固定期合同后应订无固定期限 |

### 2. 工作内容与地点

| 检查项 | 风险点 |
|-------|--------|
| 岗位描述 | 避免过于模糊（如"公司安排的工作"） |
| 工作地点 | 应具体，避免"全国"等宽泛表述 |
| 调岗权 | 单方面调岗权可能无效 |

### 3. 劳动报酬

| 检查项 | 风险点 |
|-------|--------|
| 工资结构 | 明确基本工资、绩效工资、加班费等 |
| 加班费计算 | 平日150%/休息日200%/法定假日300% |
| 最低工资 | 不能低于当地最低工资标准 |
| 支付时间 | 按月支付，不得拖欠 |

### 4. 社会保险

| 检查项 | 风险点 |
|-------|--------|
| 社保缴纳 | 必须缴纳，约定不买社保无效 |
| 缴纳地点 | 应在工作地或合同约定地缴纳 |
| 缴纳基数 | 不得低于实际工资 |

### 5. 休息休假

| 检查项 | 风险点 |
|-------|--------|
| 年休假 | 应约定按法定标准执行 |
| 病假 | 应约定按法定标准（有医院证明） |
| 请假程序 | 需明确请假审批流程 |

### 6. 解除与终止

| 检查项 | 风险点 |
|-------|--------|
| 解除条件 | 不能约定排除法定解除权 |
| 经济补偿 | N、N+1、2N等情形需明确 |
| 违法解除 | 违法解除需支付2N赔偿金 |
| 终止条件 | 不能约定，员工退休/死亡/丧失劳动能力才能终止 |

### 7. 保密与竞业限制

| 检查项 | 风险点 |
|-------|--------|
| 保密义务 | 商业秘密需明确定义 |
| 竞业限制期限 | 不超过2年（《劳动合同法》第24条） |
| 竞业限制补偿 | 必须约定补偿金，2年内按月支付 |
| 违约金上限 | 竞业限制违约金不得过高 |

---

## 风险评分标准

| 分数区间 | 风险等级 | 含义 |
|---------|---------|------|
| 1-3 | 🟢 低风险 | 符合法律要求 |
| 4-6 | 🟡 中等风险 | 部分条款不合规，需修改 |
| 7-10 | 🔴 高风险 | 重大违法，可能导致合同无效或处罚 |

## 工作流程

```
Step 1：读取合同（file tool）
  ↓ 读取 PDF/DOCX 或处理粘贴文本
Step 2：必备条款检查
  ↓ 按《劳动合同法》第17条逐项核对
Step 3：重点条款审查
  ↓ 审查试用期、工资、社保、解除、竞业等条款
Step 4：风险评分
  ↓ 对每条问题打分
Step 5：生成报告
  ↓ 按模板格式输出
  ↓ 标注 [verify] 条款
```

## 常用法律依据

- **《劳动合同法》**（2012修正）：合同订立、必备条款、解除与终止
- **《劳动合同法实施条例》**：细则规定
- **《职工带薪年休假条例》**：年休假
- **《工资支付暂行规定》**：工资、加班费
- **《关于职工工作时间的规定》**：工时制度
- **《社会保险法》**：社保缴纳
- **《职业病防治法》**：职业危害防护

---

## 注意事项

- **引用来源标注 `[verify]`**：发现违法条款时标注具体法律依据，需人工核实
- **输出标注**：所有输出必须注明"草稿，需律师审核确认"
- **区分法定与约定**：法定条款不能通过约定排除
- **地区差异**：最低工资、社会保险缴费基数等因地区而异
- **consequential actions**：发送、归档等操作需用户确认
