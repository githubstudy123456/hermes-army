# Leave Tracker · 员工假期追踪

## 核心定位

追踪员工年假、病假、事假、婚假、产假等各类假期**余额与使用情况**，支持假期政策合规性检查，输出假期余额报表与异常预警。  
适用于企业HR、法务部门对员工假期管理合规性的监控。  
输出为**草稿**，需HR/法务确认，不构成法律结论。

---

## 适用假期类型（中国大陆企业）

| 假期类型 | 法律依据 | 法定标准 | 说明 |
|---------|---------|---------|------|
| 年休假 | 《职工带薪年休假条例》 | 累计工作满1年：5天；满10年：10天；满20年：15天 | 以参加工作年限计算，不以本企业年限计算 |
| 病假 | 《劳动法》第40条 | 依医嘱，无固定天数 | 需提供医院证明 |
| 事假 | 《劳动法》 | 无明确规定，视公司制度 | 一般需批准 |
| 婚假 | 《婚姻法》 | 法定婚假+晚婚假（各地不同） | 目前晚婚假已取消 |
| 产假 | 《女职工劳动保护特别规定》 | 98天+生育奖励假（各地不同） | 含难产、多胞胎增加 |
| 陪产假 | 《人口与计划生育法》 | 各地规定不同（7-30天） | 配偶生育时可享受 |
| 丧假 | 《关于国营企业职工请婚丧假和路程假问题的规定》 | 1-3天（直系亲属） | 含路程假 |
| 工伤假 | 《工伤保险条例》 | 停工留薪期，一般不超过12个月 | 工伤认定后 |
| 哺乳假 | 《女职工劳动保护特别规定》 | 婴儿1周岁前每天1小时 | 可折算为哺乳时间 |

## 计算规则

### 年休假计算规则

```
年休假天数 = f(累计工龄)
  - 累计工作满1年不满10年：5天
  - 累计工作满10年不满20年：10天
  - 累计工作满20年：15天

注意：
- 以自然日历年度计算（每年1月1日至12月31日）
- 工龄以社保缴费记录或档案记录为准
- 跨年可结转，但不超过下一年度
- 未休年休假可获3倍工资补偿（《带薪年休假条例》第5条）
```

### 年假折算规则（入职/离职时）

```
入职当年年假折算：
  当年剩余月数 = 12 - 入职月份 + 1
  应休年假 = (年度标准年假 / 12) × 当年剩余月数（折算不足1天的舍去）

离职时未休年假补偿：
  补偿金额 = 日工资 × 3 × 未休年假天数
  日工资 = 月工资 / 21.75
```

## 追踪指标

| 指标 | 说明 | 预警阈值 |
|------|------|---------|
| 年假余额 | 当前剩余年假天数 | < 3天预警 |
| 年假使用率 | 已使用 / 年度总额 | < 50%（年度过半时）|
| 病假连续天数 | 连续病假天数 | > 30天需关注 |
| 病假累计天数 | 年度累计病假天数 | > 60天需关注 |
| 请假频率异常 | 频繁请事假/病假 | 需HR关注 |
| 假期过期预警 | 年假即将过期 | 每年12月前提醒 |

## 工作流程

```
Step 1：读取假期数据（file tool）
  ↓ 读取 HR 系统导出的假期数据 / 考勤记录
Step 2：计算假期余额
  ↓ 按法定标准计算年假、病假等各类假期余额
Step 3：合规性检查
  ↓ 检查年假计算基数、病假证明、请假审批流程
Step 4：异常识别
  ↓ 识别余额不足、连续病假、即将过期等异常
Step 5：生成报告
  ↓ 按模板格式输出
Step 6：推送（如需定时推送）
  ↓ 发送至 HR / 相关管理人员
```

## 常用法律依据

- **《职工带薪年休假条例》**（国务院令第514号）：年休假天数、补偿标准
- **《企业职工带薪年休假实施办法》**（人社部令第1号）：年休假折算、补偿计算
- **《劳动法》**（2018修正）第40条：病假、事假
- **《女职工劳动保护特别规定》**（国务院令第619号）：产假、哺乳假
- **《工伤保险条例》**（国务院令第586号）：工伤假
- **《关于国营企业职工请婚丧假和路程假问题的规定》**：婚假、丧假

---

## 注意事项

- **工龄认定**：以社保缴费记录为准，辅以档案记录
- **地区差异**：各省市生育假、陪产假等有差异，需按当地规定
- **数据隐私**：员工假期信息属个人信息，需保密处理
- **输出标注**：所有输出必须注明"草稿，需HR/法务确认"
- **consequential actions**：发送报告、归档等操作需用户确认
