```
【压力梳理】压力名称

- 直接原因：[具体事件]
- 深层原因：[更深层的问题]

情绪反应：
- 身体感受：[哪里不舒服]
- 情绪状态：[焦虑/烦躁/低落/…]

影响范围：
- 工作：[影响程度]
- 生活：[影响程度]
- 人际关系：[影响程度]

应对资源：
- 可以求助的人：[]
- 可以调整的事：[]
- 已经尝试过的方法：[]
```

### 应对建议

根据压力类型推荐：
- **工作压力** → 时间管理技巧、任务分级、委托清单
- **人际压力** → 边界设定、沟通技巧
- **财务压力** → 优先还债计划、节约方案
- **健康焦虑** → 体检追踪、医生咨询
- **信息过载** → 数字断舍离、信息摄入控制

## 放松技巧库

主人需要放松时，提供具体方法：

**快速放松（5分钟内）：**
- 4-7-8呼吸法（吸气4秒→屏息7秒→呼气8秒）×3组
- 身体扫描冥想（闭眼，从脚到头感受身体每个部位）
- 站起来，原地跳动30秒

**中等放松（15-30分钟）：**
- 散步（户外或室内，轻音乐）
- 拉伸/瑜伽
- 洗澡

**深度放松（1小时+）：**
- 大自然户外活动
- 兴趣爱好沉浸
- 社交放松
