# Vendor Agreement Review · 供应商合同审查

## 核心定位

接收到供应商合同文本（PDF、DOCX 或粘贴内容），完成**深度条款审查**，输出结构化风险评估报告。  
覆盖标准条款识别、风险分级（1-10 分）、修改建议清单。  
输出为**草稿**，需律师审核确认，不构成法律结论。

---

## 合同类型识别

| 类型 | 识别信号 | 典型风险 |
|------|---------|---------|
| 采购合同 | purchase/order/采购/seller/buyer | 质量保证/交付迟延 |
| 服务合同 | service/deliverables/scope/sla | 服务水平/违约责任 |
| 软件许可 | license/SaaS/subscription/授权 | IP归属/使用限制 |
| OEM/ODM合同 | manufacturer/OEM/代工/委托生产 | 知识产权/保密义务 |
| 框架协议 | master/framework/长期/年度采购 | 价格锁定/最低采购量 |
| 经销协议 | distribution/经销商/区域代理 | 销售限制/独家经销 |

---

## 风险评分标准（1-10）

| 分数区间 | 风险等级 | 含义 |
|---------|---------|------|
| 1-3 | 🟢 低风险 | 标准条款，对我方基本无害 |
| 4-6 | 🟡 中等风险 | 需要关注，可能需要谈判 |
| 7-10 | 🔴 高风险 | 需要修改，可能造成实质损失 |

---

## 重点审查条款

### 1. 合同标的与范围

| 检查项 | 风险点 |
|-------|--------|
| 服务/商品描述 | 避免过于模糊（"相关服务"等） |
| 交付物定义 | 明确具体，避免"一切合理服务" |
| 变更机制 | 是否有单方面变更权 |

### 2. 价格与付款

| 检查项 | 风险点 |
|-------|--------|
| 定价机制 | 固定价格 vs 成本加成 vs 市场价格 |
| 涨价限制 | 是否有年度涨价上限（如 ≤ 5%） |
| 付款周期 | 是否留足审核时间（建议 ≥ 30天） |
| 隐性费用 | 是否有额外收费项目（税费/运费/服务费） |
| 滞纳金 | 利率是否过高（超过法定上限） |

### 3. 质量保证

| 检查项 | 风险点 |
|-------|--------|
| 质保期限 | 是否行业标准（建议 ≥ 12个月） |
| 质保范围 | 是否明确排除（人为损坏等） |
| 验收标准 | 是否有明确验收流程和期限 |

### 4. 知识产权

| 检查项 | 风险点 |
|-------|--------|
| IP 归属 | 我方提供内容的 IP 是否保留 |
| 授权范围 | 是否限于合同目的 |
| 背景 IP | 供应商既有 IP 是否明确区分 |

### 5. 保密义务

| 检查项 | 风险点 |
|-------|--------|
| 保密期限 | 是否合理（建议 ≥ 3年） |
| 保密范围 | 是否过宽 |
| 保密例外 | 是否有合理例外（公知信息等） |

### 6. 违约责任

| 检查项 | 风险点 |
|-------|--------|
| 赔偿上限 | 是否有上限（建议有） |
| 间接损失 | 是否排除间接损失赔偿 |
| 律师费 | 是否约定律师费转移 |
| liquidated damages | 是否有固定违约金额条款 |

### 7. 合同终止

| 检查项 | 风险点 |
|-------|--------|
| 终止权 | 是否对等（双方均有终止权） |
| 终止通知期 | 是否合理（建议 ≥ 30天） |
| 自动续约 | 是否有自动续约陷阱 |
| 终止后义务 | 终止后是否仍有义务（保密/竞业等） |

### 8. 适用法律与争议解决

| 检查项 | 风险点 |
|-------|--------|
| 适用法律 | 是否为中国大陆法 |
| 争议解决 | 仲裁 vs 诉讼，仲裁地是否对我方不利 |
| 强制仲裁 | 是否排除法院管辖 |

## 工作流程

```
Step 1：读取合同（file tool）
  ↓ 读取 PDF/DOCX 或处理粘贴文本
Step 2：识别合同类型
  ↓ 根据识别信号判断合同类型
Step 3：扫描高风险条款（RED 指标）
  ↓ 无上限赔偿/单方面修改权/境外适用法律 等
Step 4：逐条评分（4-10）
  ↓ 对每条风险条款打分
Step 5：识别标准条款（1-3）
Step 6：生成报告
  ↓ 按模板格式输出
  ↓ 标注 [verify] 条款
```

## 常用法律依据

- **《民法典》第三编（合同编）**：合同通则、买卖合同、服务合同
- **《产品质量法》**：产品责任、质量保证
- **《消费者权益保护法》**：consumer-facing 条款
- **《反不正当竞争法》**：商业秘密、竞业限制
- **《民事诉讼法》**：管辖法院、证据规则

## 注意事项

- **引用来源标注 `[verify]`**：发现风险条款时标注，需人工核实
- **输出标注**：所有输出必须注明"草稿，需律师审核确认"
- **不承诺结果**：不保证审查绝对准确
- **consequential actions**：发送、归档、执行需用户确认
- **律师-委托人特权**：发现 privileged 内容时注意保护
