# 业务合规检查

## 常见业务合规场景

### 1. 隐私政策合规（GDPR / PIPL / 网络安全法）

| 检查项 | 要求 |
|--------|------|
| 收集告知 | 首次收集前必须明示收集目的 |
| 最小化原则 | 只收集业务必需的最小数据 |
| 存储限制 | 不得超期存储，超期必须删除 |
| 用户权利 | 提供查阅/更正/删除/撤回同意渠道 |
| 数据出境 | 需通过安全评估或个人敏感数据不得出境 |
| 儿童保护 | 14岁以下需监护人同意 |

**主要法律依据：**
- 《个人信息保护法》(PIPL) 2021
- 《网络安全法》 2017
- 《数据安全法》 2021
- 《儿童个人信息网络保护规定》

### 2. 广告法合规

| 违规词 | 风险 |
|--------|------|
| "最佳" / "第一" / "顶级" | 违法，处罚10-100万 |
| "绝对" / "100%" | 违法 |
| 虚假原价/折扣 | 违反价格法 |
| 未经授权使用肖像/名人 | 侵犯肖像权 |
| 医疗/药品广告需特殊审批 | 违反广告法 |

**主要法律依据：《广告法》2022修正**

### 3. 消费者保护

| 场景 | 规则 |
|------|------|
| 7天无理由退货 | 网购适用，实体店除外 |
| 虚假宣传赔偿 | 退一赔三，上限500元（消保法第55条）|
| 格式条款无效 | 经营者不得以格式条款限制消费者权利 |
| 个人信息不得泄露 | 消保法第29条 |

**主要法律依据：《消费者权益保护法》**

### 4. 劳动合规

| 场景 | 要求 |
|------|------|
| 试用期 | 最长6个月，不可延长 |
| 劳动合同 | 入职1个月内必须签订 |
| 社保 | 入职即缴纳，不可约定不缴 |
| 经济补偿 | N/N+1/2N三种情形 |
| 竞业限制 | 不超过2年，需按月支付补偿 |
