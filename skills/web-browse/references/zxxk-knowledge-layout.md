# 学科网(zxxk.com) 知识讲解页面结构

**验证时间：** 2026-05-27
**目标URL：** https://www.zxxk.com/zsd/
**用途：** 个人教学工具 UI 参考——只保留教学核心功能，去除广告/支付/登录

## 页面布局（简化版参考）

```
┌─────────────────────────────────────────────────────────┐
│  Logo   学科切换  搜索框                    广东/全国/中考 │
├───────────┬─────────────────────────────────────────────┤
│ 左侧导航  │  Tab: 同步 | 中考 | 知识点 | 作业 | 试卷    │
│           ├─────────────────────────────────────────────┤
│ 年级/册次 │  子Tab（知识点场景）:                          │
│           │  力学/电和磁/欧姆定律/热学/光学/             │
│ ↓ 章节树  │  电功和电功率/实验/跨学科实践/物理模型        │
│   ·知识1  ├─────────────────────────────────────────────┤
│   ·知识2  │  知识讲解内容区                               │
│   ·知识3  │  ┌──────────┐ ┌──────────┐ ┌──────────┐   │
│           │  │ 课件     │ │ 教案     │ │ 题型     │   │
│           │  └──────────┘ └──────────┘ └──────────┘   │
│           │  ┌──────────┐                               │
│           │  │ 模型     │ ← (可视化模型/动画)             │
│           │  └──────────┘                               │
└───────────┴─────────────────────────────────────────────┘
```

## 保留的核心元素

- **左侧**：年级册次选择 → 章节树 → 知识点列表（两层嵌套，年级只顶部显示一次）
- **顶部**：学科切换 + 搜索
- **中间Tab**：知识点主Tab + 子Tab按知识点分组
- **内容区**：课件/教案/题型/模型 四类资源卡片
- **模型卡片**：可视化物理模型（3D动画/交互演示）—— 这是教学价值核心

## 去除的元素

- 广告横幅
- 支付/下载积分
- 登录注册入口
- 推荐/排行榜
- 评论区

## 导航数据结构（知识点Tab）

知识点子Tab分组（以初中物理为例）：
1. 力学（声音、光的传播、透镜、凸透镜成像规律）
2. 电和磁
3. 欧姆定律
4. 热学
5. 光学
6. 电功和电功率
7. 实验
8. 跨学科实践
9. 物理模型

## 技术备注

- 域名确认：`www.zxxk.com`（`xueke.cn` 不存在）
- 技术栈：Vue.js + jQuery，多页应用
- 物理站：`wl.zxxk.com/m/`（初中）、`wl.zxxk.com/p/`（小学）
- 页面直接访问 `/zsd/` 可达知识点列表页
