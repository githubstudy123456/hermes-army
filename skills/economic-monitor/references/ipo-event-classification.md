# IPO事件分类定级规范（2026-06-12新增）

## 长鑫科技IPO注册案例

**事件**：证监会同意长鑫科技集团股份有限公司首次公开发行股票注册
**时间**：18:45（证监会）→ 18:58（新浪快讯转载）
**分类**：P2（头部企业IPO注册，非央行/货币政策）

## 定级逻辑

| IPO类型 | 例子 | 定级 | 理由 |
|---------|------|------|------|
| 巨型IPO（融资额超百亿） | 长鑫科技（半导体龙头） | P2 | 行业龙头上市影响资金面 |
| 普通企业IPO | 不知名的科创板小票 | P3-P4 | 无系统性影响 |
| 政策信号类IPO | 证监会宣布全面注册制 | P1 | 制度性变革影响市场预期 |

## 多源验证标准（IPO类）

1. **必须**：证监会官网公示（csrc.gov.cn）— 注册生效公示页
2. **辅助**：新浪财经快讯 ≥1条
3. **充分**：1官媒 + 1商业媒体 = 确认推送

**长鑫科技验证结果**：
- 源1：证监会官网 ✅
- 源2：新浪财经 18:45 ✅
- 源3：新浪快讯 18:58 ✅
- 结论：2源确认 → P2通过

## IPO事件市场影响判断模板

```
市场影响：
1. [企业简称]是国内[行业]龙头，IPO注册获批意味着即将登陆[板块]，将为[相关]板块带来新鲜血液
2. 当前市场对科技股情绪偏暖，此消息短期对[相关]板块有一定提振作用
3. 作为[国产替代/行业龙头]核心企业，[企业简称]上市将吸引机构资金关注
```

## 去重检查

同一IPO事件多源发布时，以**最早官方源时间**为事件时间，后续转载不重复推送。
今日已有报告中若含该企业名称，直接跳过。

## 不推送标准

- 港股IPO（不在A股监测范围）
- 辅导备案阶段（未到注册生效）
- 海外市场IPO（中概股除外）
