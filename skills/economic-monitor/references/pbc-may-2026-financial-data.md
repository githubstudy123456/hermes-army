## 核心数据摘要

### 一、社会融资规模存量

| 指标 | 5月末余额 | 同比增速 |
|------|----------|---------|
| 社会融资规模存量 | 458.81万亿元 | +7.7% |
| 人民币贷款（实体经济） | 277.4万亿元 | +5.5% |
| 外币贷款（实体经济） | 1.14万亿元 | -4.3% |
| 委托贷款 | 11.22万亿元 | 持平 |
| 信托贷款 | 4.67万亿元 | +7.1% |
| 企业债券 | 35.69万亿元 | +8.4% |
| 政府债券 | 100.6万亿元 | +15.1% |
| 非金融企业境内股票 | 12.43万亿元 | +4.7% |

**结构亮点**：
- 人民币贷款占社融存量60.5%（同比低1.2ppt）
- 政府债券占比21.9%（同比高1.4ppt）——财政端发力明显
- 信托贷款正增长（+7.1%）显示地产融资边际改善

### 二、前五个月社融增量

| 指标 | 前5月增量 | 同比 |
|------|----------|------|
| 社融增量累计 | 17.48万亿元 | 少增1.16万亿元 |
| 人民币贷款 | +9万亿元 | 少增1.38万亿元 |
| 外币贷款 | +1153亿元 | 多增2116亿元 |
| 企业债券 | +1.67万亿元 | 多增7577亿元 |
| 政府债券 | +5.67万亿元 | 少增6340亿元 |

**解读**：信贷少增但债券多增，企业转向直接融资；政府债券少增反映财政发债节奏放缓。

### 三、货币供应量（5月末）

| 指标 | 余额 | 同比增速 |
|------|------|---------|
| M2（广义货币） | 353.67万亿元 | +8.6% |
| M1（狭义货币） | 114.89万亿元 | +5.5% |
| M0（流通中货币） | 14.69万亿元 | +11.9% |

**注意**：M1统计口径已修订（2025年1月起），新口径包含：M0 + 单位活期存款 + 个人活期存款 + 非银行支付机构客户备付金。

### 四、前五个月人民币存款

| 指标 | 增量 |
|------|------|
| 本外币存款余额 | 352.38万亿元（+8.7%） |
| 人民币存款余额 | 344.45万亿元（+8.7%） |
| 前5月人民币存款增加 | 15.77万亿元 |
| 其中：住户存款 | +5.63万亿元 |
| 其中：非金融企业存款 | +1.26万亿元 |
| 其中：财政性存款 | +1.91万亿元 |
| 其中：非银金融机构存款 | +5.64万亿元 |

**解读**：住户存款增5.63万亿 vs 企业存款增1.26万亿，居民储蓄意愿强于企业投资意愿。

### 五、前五个月人民币贷款

| 指标 | 增量 |
|------|------|
| 人民币贷款余额 | 281.02万亿元（+5.5%） |
| 前5月人民币贷款增加 | 9.11万亿元 |
| 住户贷款 | **-6314亿元**（短期-6942亿，中长期+628亿） |
| 企业单位贷款 | +9.63万亿元 |
| 其中：短期贷款 | +3.77万亿元 |
| 其中：中长期贷款 | +4.99万亿元 |
| 其中：票据融资 | +6999亿元 |

**关键信号**：
- 住户贷款**负增长**（-6314亿）→ 居民购房/消费信贷疲软
- 企业中长期贷款+4.99万亿→ 基建/制造业金融支持仍强

### 六、银行间市场利率（5月）

| 利率类型 | 加权平均利率 | 环比 | 同比 |
|---------|------------|------|------|
| 同业拆借 | 1.31% | +0.02ppt | -0.24ppt |
| 质押式回购 | 1.33% | +0.02ppt | -0.23ppt |

**解读**：银行间市场利率同比下降（资金面宽松），但环比微升（边际收紧）。
