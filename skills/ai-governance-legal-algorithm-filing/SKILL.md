# Algorithm Filing · 算法备案

## 核心定位

依据《互联网信息服务算法推荐管理规定》（网信办令第11号，2022年）提供**算法备案全流程指导**，包括备案资格判断、材料准备、系统填报、进展追踪、变更与注销。  
专门针对网信办"互联网信息服务算法备案系统"（beian.beian.gov.cn）。  
输出为**草稿**，需律师审核确认，不构成法律结论。

## 备案适用范围

### 应当备案的主体

| 主体类型 | 说明 |
|---------|------|
| 互联网信息服务提供者 | 提供算法推荐服务的组织 |
| 算法推荐服务提供者 | 利用算法向公众提供信息服务的组织 |
| 深度合成服务提供者 | 提供深度合成服务的组织 |
| 生成式AI服务提供者 | 提供生成式AI服务的组织 |

### 备案算法类型（《算法推荐管理规定》第16条）

| 类别 | 说明 | 备案要求 |
|------|------|---------|
| 生成合成类 | 图像生成、语音生成、内容合成 | 必须备案 |
| 个性化推送类 | 个性化信息流推荐 | 必须备案 |
| 排序精选类 | 搜索结果排序、榜单 | 必须备案 |
| 检索过滤类 | 检索过滤、违规内容拦截 | 必须备案 |
| 调度决策类 | 路径规划、资源调度 | 必须备案 |

## 备案材料清单

### 通用材料（所有算法类型）

| 材料名称 | 要求 | 说明 |
|---------|------|------|
| 营业执照/法人证书 | 扫描件PDF | 加盖公章 |
| 算法主体基本信息表 | 系统在线填报 | 平台名称、服务形式等 |
| 算法信息安全管理制度文件 | PDF | 需加盖公章 |
| 算法安全评估报告 | PDF | 需加盖公章 |
| 投诉举报渠道信息 | 系统在线填报 | 公示方式、处理流程 |
| 应急处置方案 | PDF | 安全事件应急响应 |

### 分类型材料

#### 个性化推送类算法

| 材料名称 | 要求 |
|---------|------|
| 算法原理说明 | 文字说明（不少于500字） |
| 推荐机制描述 | 用户画像构建、推荐逻辑 |
| 系统部署说明 | 技术架构图 |
| 数据来源说明 | 训练数据、实时数据来源 |

#### 生成合成类算法（深度合成）

| 材料名称 | 要求 |
|---------|------|
| 生成合成内容说明 | 支持的内容类型（图像/音频/视频/文本） |
| 内容生成逻辑描述 | 模型类型、技术路线 |
| 内容标识方案 | 水印策略、标识位置 |
| 安全保障措施 | 违禁内容拦截机制 |

#### 排序精选类算法

| 材料名称 | 要求 |
|---------|------|
| 排序因素说明 | 排序因子及权重 |
| 排序结果展示逻辑 | 是否引入人工干预 |
| 公平性措施 | 防歧视、反操纵措施 |

## 备案流程

```
Step 1：判断备案资格
  ↓ 确认是否属于备案义务主体
  ↓ 确认算法类型是否在备案范围内
Step 2：材料准备（file tool）
  ↓ 准备通用材料+分类型材料
  ↓ 起草算法安全评估报告、算法信息安全管理制度
Step 3：账号注册
  ↓ 访问 beian.beian.gov.cn
  ↓ 注册账号（需实名认证）
Step 4：在线填报
  ↓ 填报算法主体基本信息
  ↓ 填报算法信息（分类型填报）
  ↓ 上传备案材料
Step 5：提交与受理
  ↓ 确认填报信息无误后提交
  ↓ 等待网信部门受理（10个工作日内）
Step 6：备案编号获取
  ↓ 审核通过后获取备案编号
  ↓ 在产品/网站显著位置展示备案编号
```

## 备案有效期与变更

| 事项 | 说明 |
|------|------|
| 有效期 | 无固定有效期，持续有效 |
| 变更备案 | 算法重大变更需30日内提交变更申请 |
| 注销备案 | 服务终止需提交注销申请 |

### 需要变更备案的情形

| 变更类型 | 说明 | 时限 |
|---------|------|------|
| 算法重大更新 | 算法逻辑、数据源重大变更 | 变更生效后30日内 |
| 服务主体变更 | 主体名称、联系方式变更 | 变更生效后30日内 |
| 服务终止 | 停止提供算法推荐服务 | 服务终止后30日内 |

## 注意事项

- **备案网址**：beian.beian.gov.cn，警惕钓鱼网站
- **备案编号公示**：获取编号后必须在产品界面显著位置公示
- **变更备案时限**：算法重大变更30日内必须提交变更申请
- **引用来源标注 `[verify]`**：法规依据需人工核实
- **输出标注**：所有输出必须注明"草稿，需律师审核确认"
- **consequential actions**：提交备案等操作需用户确认
