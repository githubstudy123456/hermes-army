# IP Assignment · 职务发明归属

## 核心定位

对员工职务发明的**知识产权归属进行认定**，评估劳动合同或协议中知识产权归属条款的合规性，识别发明人与用人单位之间的权利边界，提供归属建议。  
依据《专利法》《专利法实施细则》《著作权法》及《劳动合同法》相关规定。  
输出为**草稿**，需律师审核确认，不构成法律结论。

---

## 法律依据

| 法规 | 条款 | 核心内容 |
|------|------|---------|
| 《专利法》 | 第6条 | 职务发明与非职务发明的界定 |
| 《专利法实施细则》 | 第12条 | 执行本单位的任务、利用本单位物质技术条件 |
| 《著作权法》 | 第16条 | 职务作品的界定 |
| 《劳动合同法》 | 第22-23条 | 保密义务与竞业限制（含知识产权归属） |
| 《计算机软件保护条例》 | 第13条 | 职务计算机软件 |

## 职务发明认定标准

### 《专利法》认定标准

| 类型 | 认定条件 | 归属 |
|------|---------|------|
| 执行本单位任务 | 履行本职工作、承担单位交付的开发任务 | **归用人单位** |
| 主要利用本单位物质技术条件 | 发明创造主要依靠单位资金、设备、零部件等 | **归用人单位**（约定优先） |
| 离职1年内完成的发明 | 与原单位本职工作或任务相关 | **可能归原用人单位** |
| 非职务发明 | 非上述情形，完全利用个人时间/资源完成 | **归发明人** |

### 关键判断要素

```
判断流程：
1. 是否执行本单位任务？
   ↓ 是 → 职务发明（归单位）
   ↓ 否 → 进入下一步
2. 是否主要利用本单位物质技术条件？
   ↓ 是 → 职务发明（约定优先）
   ↓ 否 → 非职务发明（归发明人）
3. 离职1年内完成，是否与原工作相关？
   ↓ 是 → 职务发明（归原单位）
   ↓ 否 → 非职务发明
```

## 职务发明相关权益

### 发明人权利（职务发明）

| 权利类型 | 法律依据 | 说明 |
|---------|---------|------|
| 获得奖励权 | 《专利法实施细则》第78条 | 专利授权后应获得奖励 |
| 获得报酬权 | 《专利法实施细则》第78条 | 专利实施后应获得合理报酬 |
| 署名权 | 《专利法》第17条 | 发明人有权署名（不可约定排除） |

### 奖励与报酬标准

| 类型 | 标准 | 说明 |
|------|------|------|
| 发明专利奖金 | 不少于3000元/项 | 授权后一次性支付 |
| 实用新型/外观设计专利奖金 | 不少于1000元/项 | 授权后一次性支付 |
| 发明专利报酬 | 营业利润≥2%或销售收入≥0.2% | 每年实际支付 |
| 报酬计算基数 | 以发明人薪酬为基础合理折算 | 综合考虑发明人贡献 |

## 知识产权归属条款审查

### 常见条款类型

| 条款类型 | 效力分析 | 风险点 |
|---------|---------|--------|
| 约定归单位所有 | ✅ 有效 | 需明确约定在劳动合同或协议中 |
| 约定归发明人所有 | ⚠️ 仅限非职务发明 | 职务发明约定归发明人可能无效 |
| 约定分享收益 | ✅ 有效 | 约定奖励报酬分配比例 |
| 约定"一切成果归单位" | ⚠️ 效力存在争议 | 可能被认定为格式条款、显失公平 |
| 约定剥夺发明人署名权 | ❌ 无效 | 署名权是法定权利，不可约定排除 |

### 合同条款审查要点

| 检查项 | 合规要求 | 风险 |
|-------|---------|------|
| 商业秘密定义 | 需明确界定商业秘密范围 | 范围过宽可能无效 |
| 职务成果界定 | 明确哪些属于职务成果 | 过于模糊可能导致争议 |
| 奖励报酬约定 | 明确奖励报酬标准或计算方式 | 未约定则按法定标准执行 |
| 离职后义务 | 约定离职后一定期限内的保密义务 | 期限需合理 |

| 分数区间 | 风险等级 | 含义 |

## 注意事项

- **职务发明认定是核心**：依据法律规定和事实情况判断发明归属
- **署名权不可约定排除**：发明人的署名权是法定权利，任何约定排除条款无效
- **奖励报酬是法定义务**：即使合同未约定，仍需按法定标准支付
- **离职1年内完成的发明**：需关注是否与原工作相关
- **引用来源标注 `[verify]`**：法规依据需人工核实
- **输出标注**：所有输出必须注明"草稿，需律师审核确认"
- **consequential actions**：发送、归档等操作需用户确认
