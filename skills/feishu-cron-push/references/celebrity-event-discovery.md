# 明星/女团活动查询 · 平台可靠性（2026-05实测）

## 核心结论

**大麦网 (damai.cn) = 明星活动的金标准负信号。**
搜索返回 "0个商品" = 该明星近期在当地没有任何官售演出/见面会/演唱会。
这是今天发现的最可靠判断依据——其他平台全是噪音。

## 平台可靠性速查

| 平台 | 用途 | 可靠性 | 备注 |
|------|------|--------|------|
| **大麦网** | 查某明星在目标城市有无演出 | ⭐⭐⭐⭐⭐ | 0商品 = 无活动，最可靠 |
| 搜狗搜索 | 中文搜索 | ❌ | 验证码拦截（anti-spider） |
| 百度搜索 | 中文搜索 | ❌ | 验证码/滑动验证 |
| 微博超话 | 看粉丝路透/临时通告 | ⚠️ | 需登录，但可见 |
| 微信搜一搜 | 公众号文章 | ⚠️ | 搜狗反爬，同一来源 |
| 活动行 | 活动聚合 | ❌ | IP被服务器封禁 |
| 猫眼电影 | 演出/演唱会 | ⚠️ | JS渲染，需浏览器 |
| 票牛网 | 二手票/在售演出 | ⚠️ | 有票≠有活动，可能只是二手 |
| Naver (韩网) | 韩团韩网资讯 | ❌ | 中文名搜索不相关 |
| Allkpop | 韩团英文资讯 | ❌ | Cloudflare拦截 |

## 验证流程

当用户问"XX明星今天在XX城市有活动吗"时：

1. **首选：大麦网搜索**
   ```
   https://search.damai.cn/search.html?keyword={明星名}&city={城市}
   ```
   - 结果=0 → 无活动，结论明确
   - 有结果 → 读标题判断是否今天/近期

2. **大麦无结果时，再试微博/小红书**（非结构化，可能有路透或临时通告）

3. **不用试**：搜狗/百度（验证码）、活动行（封IP）、Naver（不相关）

## 大麦网页面解析

搜索结果为0时，页面关键文本：
```
搜索"宋雨琦"，共0个商品
```
直接 browser_navigate + browser_vision 即可判断，无需解析HTML。
