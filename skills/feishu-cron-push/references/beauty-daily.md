# beauty-daily.py — 颜值体系每日推送脚本

## 脚本路径
```
/home/ubuntu/.hermes/scripts/beauty-daily.py
```

## 目的
每日22:30向订阅群推送一条颜值体系内容，循环覆盖面部/身材/仪态三个维度，周日复习总结。

## 内容循环（当前砍掉穿搭和发型版）

| 星期 | 主题 | 对应文档 |
|------|------|---------|
| 周一 | 面部清洁与保湿 | 03皮肤护理详解（节选） |
| 周二 | 身材管理 | 02身材管理详解（节选） |
| 周三 | 面部防晒与美白 | 03皮肤护理详解（节选） |
| 周四 | 身材运动 | 02身材管理详解（节选） |
| 周五 | 面部抗老 | 03皮肤护理详解（节选） |
| 周六 | 仪态气质 | 06仪态气质详解（节选） |
| 周日 | 复习总结 | 随机选一篇综合要点 |

## 输出文件
```
~/.hermes/cron/output/beauty-YYYY-MM-DD.txt
```

## 架构决策

### 纯星期逻辑（优于state文件）
```python
from datetime import date
weekday = date.today().weekday()  # 0=Mon, 5=Sat, 6=Sun
```
- 不依赖 start_date 初始化
- 不依赖上次运行日期 state
- 永远与真实星期对应，不会错位

### 为什么砍掉穿搭和发型
主人明确说"穿搭和发型的可以先放一放，重点做面部，身材，仪态"。如需恢复，在 TOPICS 和 CONTENT_CYCLE 中补回索引即可。

### Cron 任务
```
Cron ID: 3623acc6233c
时间: 30 22 * * *
```
