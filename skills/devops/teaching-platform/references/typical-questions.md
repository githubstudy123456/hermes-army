## 字段规范

### 章节级 — 典型问题（放在 `小结` 之后）

| 字段 | 类型 | 说明 |
|------|------|------|
| `question` | string | 题目文本，要能用一道大题考察本章核心 |
| `分析` | string | 答案+解析，2-4句话 |
| `相关知识点` | string[] | 关联章节ID（如 `["7-1", "7-2"]`）或知识点引用（如 `["7-1-三"]`） |
| `典型模型` | string | 涉及的物理模型名称 |

**数量**：每章 2-4 道，必考章节（⭐5）建议 4 道。

### 知识点级 — 相关知识点 + 典型模型（放在知识点的 `讲解` 之后）

**相关知识点**：
```json
"相关知识点": [
  { "id": "7-2", "name": "二力平衡" },
  { "id": "7-3", "name": "摩擦力" }
]
```

**典型模型**：
```json
"典型模型": [
  {
    "name": "模型名称",
    "描述": "一句话描述核心特征",
    "应用场景": "这个模型用来解决什么问题"
  }
]
```

**何时加**：只加在重点知识点（⭐4-5 或考试高频点），普通知识点不加。

---

## 添加流程

1. 选章节（如 `5-1.json`）
2. 在 `小结` 之后加 `典型问题` 数组（2-4道）
3. 在重点知识点的 `讲解` 之后加 `相关知识点` + `典型模型`
4. Playwright 截图验证：切到"章节小结" tab → 确认典型问题显示；切到"知识讲解" tab → 确认相关知识点+典型模型显示
5. 如需更新参考 JSON，用 `python3 -c "import json; ..."` 快速修改

## 题目类型分类

| 类型 | 特征 | 例子 |
|------|------|------|
| 现象解释 | 生活场景问为什么 | "急刹车人往前冲为什么" |
| 概念辨析 | 判断说法的正误 | "静止的物体不受力，对吗" |
| 模型分析 | 给出模型问结论 | "光滑水平面上停止用力，物体怎样" |
| 综合应用 | 多知识点融合 | 结合压强+浮力分析漂浮条件 |
