## 每日cron检查清单

```
[1] browser_navigate('https://www.gov.cn/') → 获取首页快照
    检查要闻区：国常会/领导人活动
    检查最新政策区：新发文件

[2] 如有新国常会新闻：
    a. browser_navigate(国常会新闻URL) → 读取完整正文
    b. 从正文中提取：审计整改+科技汇报+教育规划+美丽中国+交通法规 等各议题
    c. 每个议题单独成条，过5关后独立推送

[3] 如有新政策文件：
    a. browser_navigate(政策文件URL) → 读取完整正文
    b. 检查今日推送目录 /home/ubuntu/.hermes/political-reports/YYYY-MM-DD/
    c. 标题相似度>70% → 静默跳过

[4] 新浪财经lid=2517作为补充（仅市场情绪，不用于政治监测）
    - 5小时内政经新闻过滤
    - 标题含SpaceX/美股/欧洲央行 → 美股/国际市场，非国内政策
```

## 国常会内容拆分原则

一次国常会新闻稿通常包含多个独立议题，正确做法：

| 议题类型 | 是否独立推送 | 理由 |
|---------|------------|------|
| 审计整改工作 | P3/P4 | 一般工作部署，归日报 |
| 全国科技大会贯彻落实汇报 | P3 | 常规汇报，归日报 |
| 《教育发展十五五规划》 | P2 | 重大专项规划，独立推送 |
| 《美丽中国建设十五五规划》 | P2 | 重大专项规划，独立推送 |
| 《道路交通安全法修订草案》 | P3 | 已于前日推送，去重 |

## P级别分类判断

- **P2核心判断**：国常会 + 专项五年规划 → P2（重要）
- **P3核心判断**：各部委一般工作部署/一般会议
- **P4核心判断**：地方政策/国际会议/一般官员活动

## 文件命名规范

```
P{级别}-{文件简称}-YYYY-MM-DD.md
例：P2-美丽中国建设十五五规划-2026-06-13.md
```
