| 条件 | 结果 |
|------|------|
| 同一事件被≥2个独立官方源确认 | ✅ 通过 |

## 第4关：事件分类评级

| 级别 | 定义 | 推送方式 |
|------|------|---------|
| P0 · 最高 | 两会/党代会/中央经济工作会开幕或闭幕 | 立即推送，全部内容 |
| P1 · 紧急 | 政治局会议、国发/国办发重大文件 | 立即推送，完整分析 |
| P2 · 重要 | 国务院常务会议、领导人重要讲话 | 立即推送，摘要分析 |
| P3 · 一般 | 各部委重要政策 | 常规推送，简短说明 |
| P4 · 低优先 | 地方政策、一般会议活动 | 归入次日日报 |

## 第5关：去重检查

- 对比今日 `~/.hermes/political-reports/` 目录下已有推送
- 标题相似度>70% → 跳过（静默）
- 新内容 → 通过

## 快速判断流程（每30分钟cron执行）

```
1. urllib直读 gov.cn/zhengce/index.htm → 提取最新政策文件列表
2. browser_navigate gov.cn 首页 → 提取时政要闻
3. Sina lid=2517（最近3小时）→ 经济政策信号补充
4. 36kr RSS → 科技商业（非官方，跳过）
5. 逐条过5关
6. 通过 → 格式化推送；未通过 → [SILENT]
```

**已验证无新文件的典型场景（正常沉默）**：
- Sina 财经 API 全部内容为市场/金融/国际新闻（美伊冲突/AI股/加密货币），无一条官方政策文件 → 5关第1关拦截，输出 [SILENT]
- 新华网政治 RSS 返回2022年旧数据，无法获取实时内容
- 搜索擎对官方域名（site:gov.cn）返回 0 结果

**这是正常状态，不是错误。** 官方政策文件发布频率本身不高（国务院每周约2-4个文件），多数扫描轮次会得出"无新内容"的结论。
