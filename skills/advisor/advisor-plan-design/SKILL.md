# 方案制定技能

> **本技能已吸收以下窄化技能的核心内容：**
> - `plan-design` — 完全相同的内容，独立维护已无必要（已归档）

## 核心功能

将模糊的目标转化为具体、可执行、有时间线的行动计划。

## 工作流程

### Step 1：确认目标和约束

明确：
- 主人想要达成什么？（目标）
- 有多少时间/资源？（约束）
- 关键里程碑是什么？

如果目标模糊，先帮主人明确化（用SMART原则）。

### Step 2：方案设计

根据目标设计 2-3 个可行方案：
- **激进方案** — 最快但风险高
- **稳健方案** — 折中，速度和安全性平衡
- **保守方案** — 最稳妥但最慢

每个方案说明：
- 执行步骤（分阶段）
- 所需资源
- 时间线
- 预期结果
- 主要风险

### Step 3：方案推荐

推荐一个最适合的方案，给出理由。

### Step 4：落地清单

把方案变成可以直接执行的动作清单：

```
【行动清单】[方案名称]

第一阶段（Day 1-3）：
☐ [具体行动]
☐ [具体行动]
☐ [具体行动]

第二阶段（Day 4-7）：
☐ [具体行动]
☐ [具体行动]

第三阶段（Week 2）：
☐ [具体行动]
☐ [具体行动]

📊 进度检查点：
- [Milestone 1]：预计完成日期 X，完成标准是...
- [Milestone 2]：预计完成日期 X，完成标准是...

⚠️ 风险预案：
如果 [可能的问题] 发生，改为 [备选方案]
```

## 方案类型模板

### 项目计划
```
目标：X个月内完成Y
阶段1（需求/规划）：Z天
阶段2（执行）：Z天
阶段3（验收/调整）：Z天
```

### 学习计划
```
目标：掌握X技能
第1步：基础学习（2周）
第2步：实践应用（2周）
第3步：深化/作品集（2周）
```

### 行动实验
```
目标：验证X假设是否成立
实验设计：[简单描述]
成功标准：[可衡量的标准]
如果成功：继续X
如果失败：改为Y
```

## 附：决策分析（Decision Matrix）

> 来源：`advisor-decision-matrix` — 本节已吸收其核心内容，原技能归档

### 多选项利弊分析矩阵

当主人需要从多个选项中做决策时使用。

**触发条件**：主人列出多个选项，要求分析利弊、推荐排序。

**分析步骤**：

1. **列出所有选项** — 明确每个选项的边界
2. **确定评估维度** — 常见维度：
   - 成本（时间/金钱/精力）
   - 收益（效果/价值）
   - 风险（失败概率/后果严重性）
   - 速度（多久见效）
   - 可持续性（长期 vs 短期）
3. **打分（1-5分）** — 每个选项在每个维度上的得分
4. **汇总加权** — 计算综合得分
5. **给出推荐** — 明确说"推荐X，因为Y"

**矩阵格式**：
```
| 选项 | 成本 | 收益 | 风险 | 速度 | 可持续性 | 总分 |
|------|------|------|------|------|----------|------|
| A    |  3   |  5   |  2   |  4   |    3     |  17  |
| B    |  4   |  4   |  3   |  3   |    4     |  18  |
| C    |  2   |  3   |  4   |  5   |    2     |  16  |
```

**推荐格式**：
```
📊 推荐：[选项B]
原因：
1. 综合得分最高（18分）
2. 可持续性最强（长期价值）
3. 风险中等，不会造成不可逆后果

如主人倾向更激进方案，可选A（收益最高但风险也最高）
如主人倾向保守，可选C（成本最低但收益有限）
```

### 复杂决策的树形分析

当选项有多个层级或条件分支时，用树形结构：

```
[目标]最大化学习效果
├── 方案A：自学
│   ├── 优势：灵活、免费
│   ├── 劣势：缺乏反馈、易卡关
│   └── 适用：自律性强、基础好
├── 方案B：AI导师（Bloom）
│   ├── 优势：苏格拉底式引导、即时反馈
│   ├── 劣势：需要配置、有学习曲线
│   └── 适用：需要系统学习、有具体目标
└── 方案C：真人导师
    ├── 优势：最高质量、个性化
    ├── 劣势：成本高、难找好老师
    └── 适用：关键转折点、有预算
```

## 重要原则

- 每个动作都是"今天就能执行"的，不是模糊的"继续努力"
- 设定检查点，不是定完计划就结束
- 如果时间/资源不足，主动说"要在X时间内完成Y，有几个地方需要取舍"
- 方案要留有余地，能应对变化
- **行动清单末尾附："确认计划后，告诉我开始执行，我会在关键节点提醒你。"**
