# Escalation Flagger · 风险升级路由

## 核心定位

接收到合同基本信息（类型、金额、风险等级、涉及问题），自动匹配**路由矩阵**，输出建议的处理路径和升级对象。  
帮助法务团队快速判断：哪些合同可以自行处理，哪些需要升级，哪些需要立即上报。  
输出为**草稿**，需律师审核确认，不构成法律结论。

---

## 路由矩阵

### 维度一：合同金额

| 金额区间 | 处理层级 | 说明 |
|---------|---------|------|
| < 50 万元 | 初级法务 | 可由初级法务自行处理 |
| 50-200 万元 | 中级法务 | 需中级法务审核 |
| 200-500 万元 | 高级法务 | 需高级法务或法务总监审核 |
| > 500 万元 | 法务总监 + CEO | 必须由法务总监审核，可选择性上报 CEO |

### 维度二：合同类型

| 合同类型 | 默认处理层级 | 特殊要求 |
|---------|------------|---------|
| NDA（标准模板） | 初级法务 | 按 GREEN 分类可快速处理 |
| NDA（非标准/复杂） | 中级法务 | 涉及境外法律/IP 归属需升级 |
| 供应商合同（标准） | 初级法务 | 按公司模板可快速处理 |
| 供应商合同（复杂） | 中级法务 | OEM/ODM/代工需升级 |
| SaaS MSA | 中级法务 | 数据处理条款需重点关注 |
| 采购合同（大额） | 高级法务 | 需财务和业务联审 |
| 劳动合同 | 中级法务（HR 法务） | 竞业限制/股权激励需升级 |
| 知识产权合同 | 高级法务 | IP 归属/许可需法务总监确认 |
| 投融资相关 | 法务总监 | 必须上报 |

### 维度三：风险等级

| 风险等级（来自其他 skill） | 升级要求 |
|------------------------|---------|
| 🟢 GREEN（低风险） | 初级法务可处理，常规流程 |
| 🟡 YELLOW（中等风险） | 中级法务审核，建议升级 |
| 🔴 RED（高风险） | 高级法务或法务总监，必须升级 |

### 维度四：问题类型

| 问题类型 | 路由目标 | 说明 |
|---------|---------|------|
| 合同条款解释 | 合同签署律师 | 原签署律师或业务对接律师 |
| 条款修改谈判 | 中级/高级法务 | 根据金额和类型确定 |
| 数据合规（PIPL/GDPR） | 隐私法务 + 高级法务 | 必须有隐私合规经验 |
| IP 归属争议 | 高级法务 + IP 法务 | 知识产权部门介入 |
| 违约/争议 | 高级法务 + 法务总监 | 根据金额确定升级层级 |
| 诉讼/仲裁 | 诉讼法务 + 法务总监 | 必须上报 |
| 监管合规 | 监管法务 + 法务总监 | 涉及外部监管必须上报 |
| 跨境/境外法律适用 | 跨境法务 + 高级法务 | 需了解境外法律 |

## 组合路由决策

### 决策规则

```
IF 合同金额 > 500万 AND 风险等级 = RED
    → 法务总监 + CEO，必须上报
    → 诉讼/仲裁途径考虑

ELIF 风险等级 = RED AND (合同类型 ∈ [IP合同, 投融资相关, 跨境合同])
    → 法务总监，必须上报

ELIF 风险等级 = YELLOW AND 合同金额 ≥ 200万
    → 高级法务审核

ELIF 风险等级 = YELLOW AND 合同类型 ∈ [SaaS MSA, 复杂供应商合同]
    → 中级法务审核，隐私/IP 问题升级

ELIF 风险等级 = GREEN AND 合同金额 < 50万
    → 初级法务处理，常规流程

ELIF 涉及数据合规问题
    → 隐私法务必须介入，无论金额

ELSE
    → 中级法务审核
```

## 工作流程

```
Step 1：接收信息
  ↓ 合同名称、类型、金额、风险等级、问题描述
Step 2：匹配路由矩阵
  ↓ 按金额 + 类型 + 风险 + 问题类型 交叉判断
Step 3：生成路由决策
  ↓ 输出处理层级 + 升级对象 + 处理路径
Step 4：输出报告
  ↓ 按模板格式输出
Step 5：建议关联 skill（如需要）
  ↓ 根据情况推荐其他 legal skill
```

## 注意事项

- **最终判断权**：路由矩阵是参考，最终处理层级由法务总监或对应律师判断
- **多人审核**：涉及金额较大或风险较高的合同，建议多人联审
- **记录存档**：所有升级决策应记录在案，供后续复盘
- **引用来源标注 `[verify]`**：发现风险条款时标注，需人工核实
- **输出标注**：所有输出必须注明"草稿，需律师审核确认"
- **consequential actions**：发送、归档、执行需用户确认
