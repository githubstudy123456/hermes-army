## 劳动合同审查

### 触发条件
- 当主人上传劳动合同文件或粘贴文本并要求"审合同"、"劳动合同审查"、"看看合同有没有问题"时自动激活

### 执行步骤
1. **读取合同（file tool）** — 读取 PDF/DOCX 或处理粘贴文本
2. **必备条款检查** — 按《劳动合同法》第17条逐项核对
3. **重点条款审查** — 审查试用期、工资、社保、解除、竞业等条款
4. **风险评分** — 对每条问题打分（1-10）
5. **生成报告** — 按模板格式输出，标注 [verify] 条款

### 法定必备条款（《劳动合同法》第17条）
| 条款 | 是否必备 | 常见问题 |
|------|---------|---------|
| 用人单位名称、住所、法定代表人 | ✅ 必备 | 信息不完整 |
| 劳动者姓名、住址、身份证件 | ✅ 必备 | 信息不完整 |
| 合同期限 | ✅ 必备 | 期限约定不明 |
| 工作内容（岗位/职位） | ✅ 必备 | 岗位描述过于模糊 |
| 工作地点 | ✅ 必备 | 地点过于宽泛（如"全国"） |
| 工作时间和休息休假 | ✅ 必备 | 与法定不符 |
| 劳动报酬 | ✅ 必备 | 报酬约定不明或低于法定 |
| 社会保险 | ✅ 必备 | 未写明或约定不买 |
| 劳动保护、劳动条件和职业危害防护 | ✅ 必备 | 条款缺失 |

### 重点审查条款
- **劳动期限**：试用期期限/工资/解除；连续订立次数
- **工作内容与地点**：岗位描述/工作地点/调岗权
- **劳动报酬**：工资结构/加班费/最低工资/支付时间
- **社会保险**：缴纳/缴纳地点/缴纳基数
- **解除与终止**：解除条件/经济补偿/违法解除
- **保密与竞业限制**：保密义务/竞业限制期限/补偿金/违约金

### 风险评分标准
| 分数区间 | 风险等级 | 含义 |
|---------|---------|------|
| 1-3 | 🟢 低风险 | 符合法律要求 |
| 4-6 | 🟡 中等风险 | 部分条款不合规，需修改 |
| 7-10 | 🔴 高风险 | 重大违法，可能导致合同无效或处罚 |

### 常用法律依据
- 《劳动合同法》（2012修正）：合同订立、必备条款、解除与终止
- 《劳动合同法实施条例》：细则规定
- 《职工带薪年休假条例》：年休假
- 《工资支付暂行规定》：工资、加班费
- 《社会保险法》：社保缴纳

## 竞业限制评估

### 触发条件
- 当主人要求"竞业限制审查"、"竞业协议评估"、"竞业条款有没有效"、"非竞争评估"时自动激活

### 执行步骤
1. **读取协议（file tool）** — 读取劳动合同或单独竞业限制协议
2. **主体适格性评估** — 判断员工是否属于可约定竞业限制的人员范围
3. **期限合规性审查** — 检查竞业限制期限是否≤2年
4. **补偿金合规性审查** — 检查补偿金是否约定、是否符合法定标准
5. **违约金合规性审查** — 检查违约金是否合理
6. **范围合理性审查** — 检查行业、地域范围是否过宽
7. **风险评分** — 对每项问题打分
8. **生成报告** — 按模板格式输出，标注 [verify] 条款

### 竞业限制法定要素
| 要素 | 标准 | 风险 |
|------|------|------|
| 适用人员 | 高级管理人员/高级技术人员/负有保密义务人员 | 普通员工约定无效 |
| 最长期限 | 不超过2年 | 超过2年部分无效 |
| 补偿金必须约定 | 不低于离职前12个月平均工资的30% | 未约定则条款无效 |
| 支付方式 | 按月支付，不得一次性支付 | 一次性预付可能被认定无效 |
| 违约金 | 与补偿金对等，通常不超过已收取补偿金的2-3倍 | 约定过高存在效力风险 |
| 行业/地域范围 | 明确具体，与实际业务覆盖范围相当 | 范围过宽可能被认定无效 |

### 常用法律依据
- 《劳动合同法》第23-24条：竞业限制的基本规定
- 《劳动争议司法解释（一）》（法释〔2020〕26号）：竞业限制补偿、违约金调整
- 《劳动合同法实施条例》：保密义务与竞业限制细则

## 员工假期追踪

### 触发条件
- 当主人要求"查假期"、"年假余额"、"请假记录"、"假期追踪"时自动激活
- 或每月定时推送假期报告

### 执行步骤
1. **读取假期数据（file tool）** — 读取 HR 系统导出的假期数据 / 考勤记录
2. **计算假期余额** — 按法定标准计算年假、病假等各类假期余额
3. **合规性检查** — 检查年假计算基数、病假证明、请假审批流程
4. **异常识别** — 识别余额不足、连续病假、即将过期等异常
5. **生成报告** — 按模板格式输出
6. **推送（如需定时推送）** — 发送至 HR / 相关管理人员

### 适用假期类型（中国大陆企业）
| 假期类型 | 法律依据 | 法定标准 |
|---------|---------|---------|
| 年休假 | 《职工带薪年休假条例》 | 累计工作满1年：5天；满10年：10天；满20年：15天 |
| 病假 | 《劳动法》第40条 | 依医嘱，无固定天数 |
| 产假 | 《女职工劳动保护特别规定》 | 98天+生育奖励假 |
| 陪产假 | 《人口与计划生育法》 | 各地规定不同（7-30天） |
| 丧假 | 《关于国营企业职工请婚丧假和路程假问题的规定》 | 1-3天（直系亲属） |
| 工伤假 | 《工伤保险条例》 | 停工留薪期，一般不超过12个月 |

### 追踪指标
| 指标 | 说明 | 预警阈值 |
|------|------|---------|
| 年假余额 | 当前剩余年假天数 | < 3天预警 |
| 年假使用率 | 已使用 / 年度总额 | < 50%（年度过半时）|
| 病假连续天数 | 连续病假天数 | > 30天需关注 |
| 假期过期预警 | 年假即将过期 | 每年12月前提醒 |

### 常用法律依据
- 《职工带薪年休假条例》（国务院令第514号）
- 《企业职工带薪年休假实施办法》（人社部令第1号）
- 《劳动法》（2018修正）第40条
- 《女职工劳动保护特别规定》（国务院令第619号）
- 《工伤保险条例》（国务院令第586号）
