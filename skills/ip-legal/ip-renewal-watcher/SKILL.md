# IP Renewal Watcher · 商标/专利续展监控

## 基本信息
- **Skill Name**: ip-renewal-watcher
- **Description**: 监控商标、专利注册有效期，在到期前30天发出续展提醒
- **Trigger**: 定期监控（每日/每周）/ 到期前30天触发
- **Category**: ip-legal

---

## 核心能力

### 1. 商标续展监控
中国商标续展流程：
- **法定期限**：注册商标有效期为10年，续展应在有效期满前12个月内申请（商标法第40条）
- **宽展期**：有效期满后6个月为宽展期，需额外缴纳延迟费
- **所需文件**：续展申请书、商标注册证复印件、主体资格证明
- **流程**：提交申请 → 形式审查 → 核准续展 → 公告

### 2. 专利续展监控
中国专利续展（仅限发明专利）：
- **法定期限**：发明专利权期限自申请日起算，实用新型/外观设计无续展制度
- **年费缴纳**：需在申请日起每年缴纳年费，逾期6个月可补缴但需缴滞纳金
- **监控重点**：年度年费到期日（每年申请日对应日）

### 3. 续展提醒逻辑
```
到期前30天 → 首次提醒（邮件/消息推送）
到期前7天  → 第二次提醒（紧迫性提示）
到期当天   → 到期预警（宽展期费用警告）
宽展期内   → 每日监控（防止失效）
```

---

## 操作步骤

### Step 1：导入知识产权清单
- 读取企业IP注册清单（Excel/CSV）
- 提取字段：注册号、类型（商标/发明专利/实用新型/外观设计）、申请日、到期日、注册人

### Step 2：计算到期日期
- 商标：申请日 + 10年（+ 6个月宽展期）
- 发明专利：追踪年费到期日（每年申请日对应日）

### Step 3：生成监控报告
- 输出格式：到期日期排序的监控清单
- 标注：剩余天数、续展状态（正常/临近/宽展/已失效）

### Step 4：推送提醒
- 提前30天推送通知到订阅群
- 包含：IP名称、注册号、到期日、续展材料清单

---

## 输入要求
- IP注册清单文件路径
- 或通过对话输入IP信息

## 输出
- 续展监控报告（Markdown表格）
- 即将到期IP清单
- 续展操作指引

---

## 注意事项（Pitfalls）
1. **宽展期遗漏**：商标宽展期仅6个月，逾期注销不可逆
2. **年费计算错误**：发明专利年费随年份递增，需核实对应年度标准
3. **国际注册**：马德里国际商标需注意原属国规定
4. **权利人变更**：续展前需确认权利人名义是否变更，可能需要同步办理变更

---

## 验证方式
- 核对清单中所有IP均有正确到期日期
- 确认提醒逻辑与到期日匹配
- 验证输出报告可读性
